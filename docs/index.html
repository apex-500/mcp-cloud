Up and Running in Minutes

Three steps to connect your AI agent to MCP Cloud.

Get your API key Step 1 
# Request a free API key
curl -X POST https://mcp-cloud-2w62.onrender.com/register \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com"}'
Call a tool Step 2 
# Get the current Bitcoin price
curl -X POST https://mcp-cloud-2w62.onrender.com/call \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"tool": "crypto_price", "args": {"symbol": "BTC"}}'
Connect via SSE (Model Context Protocol) Step 3 
# Connect any MCP-compatible AI agent
# Add to your MCP client config:
{
  "mcpServers": {
    "mcp-cloud": {
      "url": "https://mcp-cloud-2w62.onrender.com/sse?api_key=YOUR_API_KEY"
    }
  }
}
